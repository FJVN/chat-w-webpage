隐私政策

Chat w WebPage Chrome 扩展程序

最后更新：2025年12月26日

重点说明：我们不收集、存储或传输您的任何个人数据。您的 API 密钥经过加密并仅存储在您的设备上。所有 AI 对话都直接在您的浏览器和您选择的 AI 服务商之间进行。

1. 简介

Chat w WebPage（"我们"或"本扩展"）致力于保护您的隐私。 本隐私政策说明了您使用我们的 Chrome 扩展时，我们如何处理信息。

2. 数据收集

2.1 我们不收集的内容

我们明确说明我们不会做的事情：

  • 无跟踪或分析：我们不使用任何分析服务、跟踪像素或遥测数据。
  • 无用户账号：我们不需要注册，也不收集任何个人信息。
  • 无对话数据：我们从不查看、存储或传输您与 AI 模型的对话。
  • 无浏览历史：我们不跟踪您访问或分析的网站。
  • 无服务器通信：扩展不与我们的服务器通信，因为我们根本没有服务器。

2.2 保存在您设备上的数据

以下信息使用 Chrome 存储 API 本地存储在您的设备上：

  • API 密钥：AI 服务商的加密 API 密钥（OpenAI、Anthropic、Google、Azure 等）
  • 配置信息：您的模型设置、助手配置和偏好设置
  • 对话历史：为方便使用而本地存储的聊天记录（您可以随时清除）
  • 自定义助手：您的自定义助手提示词和模板

所有这些数据都使用 chrome.storage.local 存储，永远不会离开您的设备。

3. API 密钥安全

3.1 加密

您的 API 密钥在存储到 Chrome 本地存储之前会被加密。我们使用行业标准的加密方法来保护您的凭据。

3.2 直接通信

当您与 AI 模型聊天时，通信直接在您的浏览器和 AI 服务商的 API（OpenAI、Anthropic、Google 等）之间进行。 您的 API 密钥仅用于验证这些请求。我们从不拦截、存储或访问这些通信。

4. 第三方服务

Chat w WebPage 作为第三方 AI 服务的客户端界面。当您使用本扩展时，您需要遵守所选 AI 服务商的隐私政策：

  • OpenAI：OpenAI 隐私政策
  • Anthropic：Anthropic 隐私政策
  • Google (Gemini)：Google 隐私政策
  • Microsoft (Azure)：Microsoft 隐私声明

我们建议您查看这些政策，以了解每个服务商如何处理您的对话。

5. 权限说明

我们的扩展请求以下 Chrome 权限：

  • storage：在您的设备上本地保存您的设置和 API 密钥
  • activeTab：从您当前查看的网页中提取内容
  • sidePanel：在 Chrome 侧边栏中显示聊天界面
  • tabs：访问网页信息以提取内容
  • scripting：将内容提取脚本注入网页
  • <all_urls>：允许向您选择的 AI 服务商（OpenAI、Anthropic、Google、自定义端点等）发起 API 调用

这些权限仅用于扩展的核心功能，不用于跟踪或数据收集。

6. 数据保留

扩展存储的所有数据都保留在您的设备上，直到您：

  • 手动清除对话历史
  • 删除 API 配置
  • 卸载扩展
  • 清除 Chrome 的扩展数据

卸载扩展将自动删除所有本地存储的数据。

7. 儿童隐私

我们的扩展不面向 13 岁以下的儿童。我们不会故意收集儿童的信息。如果您是家长或监护人， 并认为您的孩子使用了本扩展，请注意我们不会收集或传输任何数据。

8. 透明度

Chat w WebPage 重视透明度。您可以在以下位置查看我们的文档和官网代码： GitHub 仓库

9. 政策变更

我们可能会不时更新本隐私政策。任何更改都将发布在此页面上，并更新"最后更新"日期。 我们建议您定期查看本政策。

10. 您的权利

由于所有数据都本地存储在您的设备上，您拥有完全控制权：

  • 访问：您可以在扩展的设置中查看所有存储的数据
  • 修改：您可以编辑或更新任何存储的信息
  • 删除：您可以随时清除所有数据
  • 导出：您可以以各种格式导出对话历史

11. 法律合规

本扩展符合：

  • 欧盟通用数据保护条例（GDPR）
  • 加州消费者隐私法（CCPA）
  • Chrome Web Store 开发者计划政策

由于我们不收集个人数据，大多数数据保护要求不适用于我们的服务。
